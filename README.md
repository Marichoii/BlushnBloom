## Overview

**Blush & Bloom** is a web application designed to suggest makeup looks tailored to specific occasions.  
By selecting a scenario — such as a job interview, date night, or casual outing — users receive curated suggestions that include color palettes, product tips, and styling advice.

This project explores the intersection of beauty, user experience, and responsive design, with a soft and approachable aesthetic inspired by floral themes.

## Features

- Occasion-based makeup recommendations  
- Curated color palettes and suggested products  
- Responsive interface using Bootstrap 5  
- Lightweight, clean front-end implementation  
- Friendly and elegant floral UI concept

## Technologies Used

- **HTML5** – structure and semantics  
- **CSS3** – custom styling and animations
